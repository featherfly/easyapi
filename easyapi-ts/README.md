## Platform Support

### web

version end with number implement with axios, such as `0.2.2`.

### uniapp

version end with char u implement with uniapp, such as `0.2.2-u`.

## Installing

### Package manager

Using npm:

```bash
$ npm install @yufei-featherfly/easyapi-ts
```

Using bower:

```bash
$ bower install @yufei-featherfly/easyapi-ts
```

Using yarn:

```bash
$ yarn add @yufei-featherfly/easyapi-ts
```

Using pnpm:

```bash
$ pnpm add @yufei-featherfly/easyapi-ts
```
